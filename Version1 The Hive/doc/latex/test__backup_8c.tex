Sauvegarde les informations sur le joueur, son inventaire, son équipement ainsi que les informations sur la carte et les quêtes d\textquotesingle{}une partie test. 

Informations sur le joueur \+: points de vie, points d\textquotesingle{}énergie, points d\textquotesingle{}action, position sur la carte, nombre de tours restants

Informations sur la carte \+: pour chaque case de la matrice map, on sauvegarde son type, sa catégorie, s\textquotesingle{}il y a un combat, si le joueur a déjà fouillé la case et si une quête y est positionnée.

Informations sur les quêtes \+: on sauvegarde les coordonnées de chaque quête ainsi que leurs états.
